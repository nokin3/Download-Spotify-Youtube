สรุปสิ่งที่ต้องมีเลยนะ
1. python
2. yt-dlp.exe
3. ffmpeg.exe & dll & other exe
4. bat ไฟล์ที่ผมททำไว้ช่วยโหลด load-song-spotify.bat , vdo-best-avc.bat , youtube-mp3.bat 3 อันนี้ อันแรกโหลดแพง อันกลางโหลด vdo youtube อันท้ายโหลด youtube เป็น mp3


-----------------------------------------------------------------------------------------------------


วิธีการใช้งาน
ลง python ตามข้อ 1
โหลดโฟลเดอร์ผมออกไป
กดไฟล์ install.bat รอมันลงเสร็จ แค่นี้ก็เสร็จแล้ว
กดไฟล์ load-song-spotify.bat ใส่ลิ้งเพลง spotify > enter รอมันโหลด ก็ได้เพลงแล้ว
กดไฟล์ vdo-best-avc.bat ใส่ลิ้งจาก youtube ก้ได้ vdo youtube
กดไฟล์ youtube-mp3.bat ใส่ลิ้งจาก youtube ก้ได้ mp3
